В структурной разметке тэги используются для описания логического представления документа, а описание визуального представления хранится отдельно (как правило, в стилевых таблицах).

Текст может быть структурирован при помощи следующих тэгов: < h1>…< h6>, < em>, < strong>, < ins>, < del>, < q>, < blockquote>, < dfn>, < abbr>, < acronym>, < code>, < samp>, < var>

Примеры:

Заголовок 1

Заголовок 2

Заголовок 3

Заголовок 4

Заголовок 5
Заголовок 6
курсив, жирный курсив цитата
